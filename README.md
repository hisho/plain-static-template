# plain-static-template
# NODE VERSION >=14.8.0 AND USE NPM

## features
- React SSG
- Image Optimize
- Generate webp
- svg sprite

## Browser
- Chrome 
- FireFox
- Safari

## Command

### Install

```shell
$ npm ci
```

### Watch
##### watch when development

```shell
$ npm start
```

##### watch when production

```shell
$ npm run start-pro
```

### Build

##### build when development

```shell
$ npm run build-dev
```

##### build when production

```shell
$ npm run build
```

### Build & Watch

```shell
$ npm run serve
```
